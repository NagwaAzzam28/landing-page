## skills practiced

- Building a structured HTML page.
- Creating layout with Flexbox.
- working with images and resizing them.
- Working with margin and padding.
- Using Git and gitHub.
- working with the Linux command line.

## what i learned

- Recreating a webpage from an image

- More CSS properties like border-radius and padding-block
- Linking and using external font from google fonts.
